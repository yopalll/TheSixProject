# 🏷️ F04: Category System

> CRUD Kategori & Sub-kategori oleh Admin.

---

## Fitur

1. **Category CRUD** — Create, Read, Update, Delete kategori
2. **SubCategory CRUD** — Create, Read, Update, Delete sub-kategori (linked to category)
3. **Category Image** — Upload icon/image untuk setiap kategori
4. **Status Toggle** — Active/Inactive status per kategori
5. **Slug Generation** — Auto slug dari nama
6. **Dynamic Subcategory Loading** — AJAX load subcategories by category ID
