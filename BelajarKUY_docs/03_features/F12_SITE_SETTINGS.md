# ⚙️ F12: Site Settings

> Pengaturan situs, SMTP, dan partner management.

---

## Fitur

1. **Site Info** — Logo, nama situs, phone, email, address, social media (key-value pairs)
2. **SMTP Settings** — Mail host, port, username, password, encryption
3. **Midtrans Settings** — Server key, client key, production mode, merchant ID
4. **Google OAuth Settings** — Client ID, client secret, redirect URL
5. **Slider Management** — Hero slider images + text
6. **Info Box Management** — Value proposition boxes
7. **Partner Management** — Partner logos
